# km-i18n-manager&#x20;

## 适用场景

- 需要给页面或组件增加多语言支持
- 需要新增翻译 key、补齐缺失翻译或新增语言
- 需要让路由标题、tabbar 文案或组件私有文本接入 i18n

## 使用方式

直接说明以下信息即可：

- 目标应用，例如 `example`
- 要处理的页面、组件或模块
- 需要支持哪些语言
- 具体要新增或修改哪些文案

示例：

```text
在 example 应用里给用户中心页面增加中英文支持，
并把 tabbar 的“我的”也接入 i18n。
```

## 结果

通常会修改：

- `apps/<app>/src/locales/lang/*.json`
- `apps/<app>/src/locales/index.ts`
- 目标页面或组件文件

必要时会补充：

- 路由标题或 tabbar 文案中的 `$t()` / `t()` 调用
